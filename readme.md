# Whales Club AMBR Weekly Drop for Rare NFTs

## Rules

1. 1 AMBR = 0.1 TON
2. 1 RS = 0.01 AMBR
3. Only whales of type: { tale: 'ton cashback' | 'whales tag' } or { tier: 'legendary' } or { sex: 'female' } is eligible for drop
4. Drop every week

## How to use

```npm i```

```npm run drop```
